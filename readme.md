**Тестовое задание. Необходимо было сделать аналог ленты историй как в инстаграмме/вк. Архитектура MVVM + SwiftUI**

*Сама задача:*
- Ленту аватарок с пагинацией
- Открытие экрана при тапе на одну из аватарок
- Отображение фотографии в фулл сайз
- Реализовать базовый функционал лайка
- При открытии фотографии необходимо было запомнить, что ее открывали и менять цвет обводки опираясь на эти данные

*Setup:*
Используются сторонние библиотеки. Для установки зависимостей в корне проекта вызови терминал и пропиши pod install. 
После установки подов запусти проект через Stories.xcworkspace
